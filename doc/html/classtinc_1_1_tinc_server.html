)
overridevirtual

Initiate a barrier by notifying all connections to lock, and then unlock after all acknowledge.

Parameters
groupgroup to make the barrier for. 0 is all.
timeoutsectimeout in seconds
Returns
true if all connections locked and unlock message was sent before the timeout

This requires clients to explicitly call barrier() . This is used to syncrhonize applications so they resume execution after all have finished executing a piece of code
